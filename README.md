# 📝 inrudder TODO

日付サイドバー・カテゴリ管理・持ち越し機能を備えたシンプルなTODO管理アプリです。

🔗 **[Live Demo](https://jun1slyme.github.io/inrudder/todo.html)**

---

## ✨ 機能

- **日付サイドバー** — 日付ごとにタスクを整理。クリックで切り替え
- **カテゴリ分類** — 「採用」「inrudder」「プライベート」の3カテゴリでタスクを色分け管理
- **タスク持ち越し** — 未完了タスクは翌日へ自動で引き継がれる
- **インライン編集** — タスク名をその場で直接編集
- **メモフィールド** — タスクごとに詳細メモを追加可能
- **ドラッグ＆ドロップ** — デスクトップ・モバイル両対応の並び替え
- **未完了バッジ** — サイドバーに残タスク数をバッジ表示
- **完了アニメーション** — タスク完了時に紙吹雪エフェクト 🎉
- **モバイル対応** — iPhone・スマートフォンでの利用を考慮したレスポンシブデザイン

---

## 🚀 使い方

ブラウザで `todo.html` を開くだけで動作します。インストール不要。

```bash
# ローカルで開く場合
open todo.html
```

または [GitHub Pages のデモ](https://jun1slyme.github.io/inrudder/todo.html) をそのまま使用できます。

---

## 🛠 技術スタック

- HTML / CSS / JavaScript（バニラJS、依存なし）
- ローカルストレージでデータを保存
- GitHub Pages でホスティング
